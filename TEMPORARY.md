# Тимчасові блоки (короткий реєстр)

Нижче — перелік тимчасових рішень, які мають бути видалені або замінені. Для кожного: причина існування, умова видалення, дедлайн/тригер.

## TEMP-001 — Перехідна нормалізація часу барів

- Локація: [core/time_geom.py](core/time_geom.py)
- Причина: перехідний шар, щоб API віддавав end‑incl без міграції SSOT на диску.
- Умова видалення: всі writer-и SSOT пишуть end‑incl, а API більше не потребує normalize_bar.
- Дедлайн/тригер: після міграції SSOT на end‑incl і завершення перевірок інваріанту часу.

## TEMP-002 — Legacy endpoint /api/latest

- Локація: [ui_chart_v3/server.py](ui_chart_v3/server.py#L630-L670)
- Причина: сумісність із старими клієнтами, що опитують /api/latest.
- Умова видалення: відсутні клієнти /api/latest за спостереженням і всі споживачі перейшли на /api/updates.
- Дедлайн/тригер: після 2 тижнів без звернень у логах до /api/latest.
